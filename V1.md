4. **TLS on the proxy → kill cleartext.** Front the k3s nginx / Tailscale path with TLS
   (Tailscale Serve gives free HTTPS at a MagicDNS name), standardize on the hostname, then flip
   Android `network_security_config.xml` `cleartextTrafficPermitted="false"`. Closes the known,
   accepted Phase-0 hole. (ROADMAP #1.)
5. **Decide the token-in-cloud-backup question.** The Android LLAT (a full house credential,
   incl. locks) is included in Google cloud backup + device-transfer by
   `backup_rules.xml`/`data_extraction_rules.xml`, and `CredentialStore` is unencrypted DataStore.
   Pick one: (a) exclude from backup (re-login on a new phone), or (b) Keystore-wrap the token.
   **Owner decision needed** — see Open decisions.
6. **OAuth to HA** — replace the long-lived token with HA's OAuth + refresh, web + Android
   together so the token story stays one story. (ROADMAP #2.) The largest single item here; see
   Open decisions on whether it blocks v1.
